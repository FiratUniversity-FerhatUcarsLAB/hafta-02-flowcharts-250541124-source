User: Kart takılır ve PIN sorulur
Assistant: Bu adımlar ATM para çekme işleminin algoritması veya akış diyagramı için tipik bir senaryodur. Aşağıda hem algoritma biçiminde hem de akış diyagramı mantığıyla açıklanmış hâlini bulabilirsin.

Assistant (Algoritma):
1. Kart takılır.
2. PIN sorulur.
3. PIN doğruluğu kontrol edilir.
   - Eğer PIN yanlışsa, hatalı giriş sayısını 1 artır.
   - Hatalı giriş sayısı 3 olursa, kart bloke edilir ve işlem sonlandırılır.
   - Doğru PIN girildiyse devam et.
4. Bakiye sorgulanır.
5. Çekilmek istenen tutar girilir.
6. Yetersiz bakiye kontrolü yapılır.
   - Eğer bakiye yetersizse “Yetersiz bakiye” mesajı ver ve tekrar giriş iste.
7. Tutarın 20 TL'nin katı olup olmadığı kontrol edilir.
   - Değilse “Tutar 20 TL'nin katı olmalıdır” mesajı ver ve tekrar giriş iste.
8. Günlük limit kontrolü yapılır.
   - Limit aşılmışsa “Günlük limit aşıldı” mesajı ver ve tekrar giriş iste.
9. Para verilir.
10. Fiş çıkarılır.
11. Kullanıcıya sor: “Başka işlem yapmak ister misiniz?”
    - Evet → Adım 4'e dön (bakiye sorgulama).
    - Hayır → Kart iade edilir, işlem sonlandırılır.

User: pseudocode şeklinde yaz
Assistant: (Provides pseudocode in English-like pseudocode)
BEGIN
  INSERT card
  ATTEMPTS ← 0
  REPEAT
      ASK "PIN giriniz: "
      READ pin
      IF pin == correct_pin THEN
          access_granted ← TRUE
      ELSE
          ATTEMPTS ← ATTEMPTS + 1
          DISPLAY "Hatalı PIN!"
      ENDIF
  UNTIL access_granted == TRUE OR ATTEMPTS == 3
  IF ATTEMPTS == 3 THEN
      DISPLAY "Kart bloke oldu."
      EJECT card
      END
  ENDIF
  REPEAT
      DISPLAY "Bakiyeniz: ", balance
      ASK "Çekmek istediğiniz tutarı giriniz: "
      READ amount
      IF amount > balance THEN
          DISPLAY "Yetersiz bakiye!"
      ELSE IF amount MOD 20 != 0 THEN
          DISPLAY "Tutar 20 TL'nin katı olmalıdır!"
      ELSE IF amount > daily_limit THEN
          DISPLAY "Günlük limit aşıldı!"
      ELSE
          balance ← balance - amount
          DISPLAY "Para veriliyor..."
          PRINT "Fiş çıkartılıyor..."
      ENDIF
      ASK "Başka işlem yapmak ister misiniz? (E/H): "
      READ answer
  UNTIL answer == "H"
  DISPLAY "Kartınızı alınız."
  EJECT card
END

      BİTİR
  SON
  TEKRAR ET
      YAZ "Bakiyeniz: ", BAKİYE
      YAZ "Çekmek istediğiniz tutarı giriniz: "
      TUTAR_OKU(tutar)
      EĞER tutar > BAKİYE İSE
          YAZ "Yetersiz bakiye!"

          YAZ "Para veriliyor..."
          YAZ "Fiş çıkarılıyor..."

      YAZ "Başka işlem yapmak ister misiniz? (E/H): "
      CEVAP_OKU(cevap)
